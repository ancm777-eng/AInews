1. Anthropic 'Claude Code' Massive Source Code Leak and Discovery of Critical Agent Autonomy Bypass Vulnerability

📅 2026-03-31 🏢 SecurityWeek Paradigm Shift: Security
🔭 Overview

(Reason for Exception: Based on Anthropic's source code leak on March 31, 2026, a highly critical zero-day prompt injection vulnerability capable of remotely bypassing core security controls was disclosed on April 1. This is exceptionally included due to its immediate and massive impact on the current enterprise agent security ecosystem.)

Following the leak of the core source code of Anthropic's agentic coding assistant, Claude Code, onto the public internet, a critical security vulnerability exploiting this was subsequently discovered, issuing a severe warning to the global software supply chain. This incident originated from a simple packaging management oversight where a debugging source map (.map) file was not removed during the deployment of Claude Code version 2.1.88 to the npm registry. Consequently, a 59.8MB JavaScript source map file containing approximately 512,000 lines of unobfuscated TypeScript source code comprising approximately 1,900 files was fully exposed.

More critically, Adversa AI reported a vulnerability capable of significantly weakening Claude Code's permission control system. To prevent system UI lag, Claude Code had a hardcoded limit capping command analysis at 50 subcommands. If an attacker tricks the agent into executing a pipeline containing more than 50 subcommands, the system ignores 'Deny' rules and falls back to an 'Ask' state, allowing covert exfiltration of SSH private keys and AWS credentials. This signifies a new security paradigm collapse in the era of autonomous agents.

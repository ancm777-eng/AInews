1. NVIDIA Releases Gated DeltaNet-2, a Linear Attention Layer Decoupling Erase and Write Operations

🔭 Overview

NVIDIA has officially released Gated DeltaNet-2, a highly advanced linear attention layer designed to address the fundamental memory bottlenecks of modern transformer architectures.

Traditional softmax attention requires an unbounded key-value (KV) cache, which leads to quadratic complexity and massive memory consumption during long-sequence generation. Linear attention models attempt to solve this by using a fixed-size recurrent state, but editing this compressed memory without scrambling existing associations has remained a significant technical challenge. Gated DeltaNet-2 overcomes this bottleneck by decoupling the active memory edit into two distinct channel-wise gates: an erase gate and a write gate, operating under a refined delta rule. Trained at 1.3 billion parameters on 100 billion FineWeb-Edu tokens, the architecture has demonstrated superior performance across standard research benchmarks, outperforming established models such as Mamba-2, Gated DeltaNet, KDA, and Mamba-3. This release represents a major paradigm shift in model architecture, offering a highly efficient, constant-memory decoding alternative for long-context AI workloads.
